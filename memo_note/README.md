# 📝 파이썬 메모장 (Python Memo Note)

간단한 커맨드 라인 인터페이스(CLI) 기반의 파이썬 메모장 애플리케이션입니다. 사용자는 터미널 환경에서 간편하게 메모를 작성, 조회 및 검색할 수 있습니다.

## 📌 주요 기능

1.  **메모 쓰기 (Write)**
    *   사용자가 입력한 내용을 현재 날짜 및 시간(`YYYY-MM-DD HH:MM:SS`)과 함께 저장합니다.
    *   데이터는 `memo.txt` 파일에 누적되어 저장됩니다.

2.  **메모 읽기 (Read)**
    *   지금까지 저장된 모든 메모 목록을 불러와서 출력합니다.
    *   저장된 메모가 없을 경우 알림 메시지를 표시합니다.

3.  **메모 검색 (Search)**
    *   특정 키워드가 포함된 메모를 찾아 보여줍니다.
    *   검색된 결과의 개수도 함께 확인할 수 있습니다.

## 📂 파일 구조

*   **`main.py`**: 프로그램의 시작점입니다. 사용자에게 메뉴(UI)를 제공하고 입력을 처리하여 `memo_manager.py`의 기능을 호출합니다.
*   **`memo_manager.py`**: 핵심 로직이 담긴 모듈입니다. 파일 입출력(I/O)을 통해 메모를 저장(`save_memo`), 읽기(`read_memos`), 검색(`search_memo`)하는 함수들을 포함합니다.
*   **`memo.txt`**: 실제 메모 데이터가 텍스트 형식으로 저장되는 파일입니다. (프로그램 실행 시 자동 생성되거나 갱신됩니다)

## 🚀 실행 방법

### 1. 필수 조건
*   Python 3.x 환경이 설치되어 있어야 합니다.

### 2. 실행 명령어
터미널에서 프로젝트 디렉토리로 이동한 후 아래 명령어를 입력하세요.

```bash
python main.py
```

## 💻 사용 예시

프로그램을 실행하면 다음과 같은 메뉴가 나타납니다.

```text
--- 📝 파이썬 메모장 ---
1. 쓰기 / 2. 읽기 / 3. 검색 / 4. 종료
원하는 메뉴를 선택하세요: 
```

*   **1번**을 선택하여 메모를 기록합니다.
*   **2번**을 선택하여 전체 메모를 확인합니다.
*   **3번**을 선택하여 특정 단어를 검색합니다.
*   **4번**을 선택하여 프로그램을 종료합니다.
